АВТОСЕРВИС
----------------------------------------------
КЛИЕНТ создает учетную запись (марка авто. пробег.год выпуска. номер телефона),создает авто в личном гараже.
Просматривает историю своих заказов. Доступны текущие акции сервиса.

ЗАКАЗ содержит данные клиента,наименование работ,дату подачи,,авто,ориентировочную стоимость.
АКЦИЯ содержит информацию об акционных предложениях(дисконт, дата проведения).
МЕНЕДЖЕР создает клиента и заказы.Может дополнять заказы доп работами, видит текущие акции.
АДМИНИСТРАТОР управляет КЛИЕНТАМИ ЗАКАЗАМИ,МЕНЕДЖЕРАМИ,АКЦИЯМИ.

НЕЗАРЕГИСТРИРОВАННЫЙ ПОЛЬЗОВАТЕЛЬ может просматривать информацию о ценах в автосервисе.
